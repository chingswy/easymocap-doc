# 调参心得

- 好的2D胜于算法调参
- 对于当前不能解决的问题，需要确保在当前的2D的情况可以解决这个问题
- 首先要不考虑速度的去解决问题，在确保跑满性能的情况下问题可以解决，再考虑速度提升
- 对于一些小问题，首先确保可以通过全手动的方式解决，再逐步替换掉其中可以自动化的步骤。
- 对于时序的输入，先确认在单帧的输入下没问题，再延伸到时序的
